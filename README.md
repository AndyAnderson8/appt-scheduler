# Appointment Scheduling Application

## Description
University coursework project demonstrating a Java-based CRUD application with user authentication, database-backed persistence, and a JavaFX user interface.

The application was designed to interact with a MySQL database for managing customers and appointments. While the original database instance is no longer available, the codebase is preserved to showcase application structure, data access patterns, and UI logic.
